Basic Questions Every Aspiring Full-Stack Developer Must Know

1. What is Internet?

Answer: The Internet is a global network that connects millions of computers to share information.

Example: When you search something on Google, you're using the Internet.

2. What is Browser?

Answer: A browser is software that helps you access and view websites.

Example: Google Chrome, Firefox, Safari.

3. What is URL?

Answer: URL (Uniform Resource Locator) is the address used to visit a website.

Example: https://www.google.com

4. What is Website?

Answer: A website is a collection of related web pages hosted online.

Example: https://www.amazon.com

5. What is Web App?

Answer: A web app is an interactive website where users can perform tasks.

Example: Google Docs lets you write and edit documents online.

6. What is Domain, Subdomain, and Main Domain?

Domain: The name of a website that people type to visit.

Example: example.com

Subdomain: A part added before the main domain for a section of the site.

Example: blog.example.com

Main Domain: The primary domain without any subdomain.

Example: example.com

8. What is IP Address?

Answer: An IP address is a unique number for a device on a network.

Example: 192.168.1.1

9. What is Bug?

Answer: A bug is a mistake or error in a program that causes it to not work properly.

Example: A login page not opening due to wrong code.

10. What is Debug?

Answer: Debugging is the process of finding and fixing bugs in a program.

Example: A developer fixing code that crashes an app.

11. What is HTML1 to HTML5?

Answer: HTML1 to HTML5 are versions of HTML. HTML5 is the latest with more features.

Example: HTML5 supports video and audio tags, HTML1 did not.

12. What is XHTML?

Answer: XHTML is a stricter version of HTML that follows XML rules.

Example: All tags in XHTML must be properly closed like <br />.

13. Difference Between Programming Language and Markup Language?

Answer: A programming language writes logic (actions), a markup language defines structure.

Example: Python is a programming language; HTML is a markup language.
